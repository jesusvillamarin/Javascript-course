Hay 3 formas de declarar un metodo en Javascript

Básica
funtcion miMetodo(base,altura){...}
Arrow function (Cuando es solo una linea de código)
let miMetodo = (base,altura) => base*altura/2
Arrow function varias lineas de codigo
let miMetodo = (base,altura) => { base*altura/2; base=0; altura=0; }
